# Конфигурационные файлы сервера

## Используемый файл

**`.htaccess`** - конфигурация для Apache серверов
- ✅ Активно используется
- ✅ Содержит настройки кэширования и сжатия
- ✅ Настроены правила rewrite

## Альтернативные конфигурации (не используются)

Если вы используете другой веб-сервер, вы можете использовать один из следующих файлов:

### Nginx

Файл `nginx_rmwebsite.conf` содержит пример конфигурации для Nginx. Скопируйте правила в ваш основной конфигурационный файл Nginx.

### IIS (Windows Server)

Файл `web.config` содержит конфигурацию для IIS с URL Rewrite модулем.

## Рекомендация

- Если используете Apache - используйте `.htaccess` (уже настроен)
- Если используете Nginx - используйте правила из `nginx_rmwebsite.conf`
- Если используете IIS - используйте `web.config`

**Важно**: Используйте только конфигурацию для вашего сервера. Неиспользуемые файлы можно безопасно удалить или переместить в архив.
